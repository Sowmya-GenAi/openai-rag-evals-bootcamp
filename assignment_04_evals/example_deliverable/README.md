# Example Deliverable — Assignment 4

This folder is **an exemplar**, not real run output. It shows what a passing
Assignment 4 deliverable looks like so you know the bar before you start.

Files:

- `comparison.md` — the 1-page comparison report (the main graded artifact).
- `eval_baseline_summary.json` — abbreviated example of the JSON report your
  harness produces. The real reports include all 21 items per file with full
  retrieval + judge data; this excerpt only includes the metric block to keep
  the example readable.
- `eval_large_chunks_summary.json` — the same shape, for the second config.

**Do not check your real run outputs into this folder.** Your runs go to
`reports/` at the repo root.
